# ssp-server
ssp-server - серверная часть системы SSP (Secured SeamlessTCP Proxy).
Сервис управляет всеми сессиями и служит единственной точкой выхода 
для всех проксируемых соединений в системе.

## Конфигурация
`host` - адрес для привязки

`timeout` - максимальное время ожидания ответа при передаче данных

`log_level` - уровень логирования
